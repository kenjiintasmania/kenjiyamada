# 先生向け：成績の集約（スプレッドシート連携）

3アプリ（単語・模擬テスト・英検）の成績を、先生のスプレッドシートに集めるしくみです。

## しくみ

- 生徒が **マイページ**（`/me/`）を開くと、その端末の最新（最高）記録が
  先生のスプレッドシートへ自動送信されます。
- 送信先は **英検アプリと同じGAS**（＝同じスプレッドシート）。
  English Loop（英検）アプリの1回ごとの記録もこれまでどおり残ります。

## セットアップ（1回だけ）

1. 記録を集めたいスプレッドシート（英検の記録が入っている既存のもの）を開く。
2. メニュー **［拡張機能］→［Apps Script］** を開く。
3. 既存のコードを **`score_gas.gs` の中身にまるごと置きかえて** 保存。
   - ※元のコードはコピーして控えておくと安心です。
4. **［デプロイ］→［デプロイを管理］** で、既存のウェブアプリを
   **［編集（鉛筆）］→ バージョン「新バージョン」→［デプロイ］**。
   - これで **URLは変わらず** 中身だけ新しくなります（アプリ側の変更は不要）。
5. 初回は権限の承認を求められたら許可してください。

これで、次の2つのタブが自動で作られ、更新されていきます。

| タブ | 内容 |
|------|------|
| **成績まとめ** | 生徒1人につき1行。3アプリの最新（最高）記録を横に並べて集約。 |
| **英検テスト履歴** | 英検アプリの1回ごとの結果を追記（従来の記録）。 |

## メモ

- 「成績まとめ」は **組＋番号** で1人1行を判別し、上書き更新します。
  単語数・模試の点・英検レベルなどの記録列は **下がらない（最高値を保持）** ようにしています。
- もし既存の英検ログを別のタブ名で続けたい場合は、`score_gas.gs` の
  `EIKEN_SHEET` をそのタブ名に合わせてください。
- 送信先URLは各アプリのHTML内 `GAS_URL` / `SUMMARY_GAS_URL` に書かれています。
  デプロイURLを作り直した場合は、そこも更新してください。
