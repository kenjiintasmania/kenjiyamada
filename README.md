# 単語アプリ（基本編＆拡張編）

英単語を**品詞ごと**に練習できる、スマホ対応の単語アプリです。
ブラウザだけで動き、サーバーは不要。クリア状況は端末に保存されます。

👉 `index.html` をブラウザで開くだけで使えます。

---

## 収録単語（合計ちょうど 2000 語）

| モード | 語数 | 出典 |
|---|---|---|
| 📕 **基本編** | 435 語 | 先生の単語集（`my_words.xlsx`）の品詞別タブから抜粋 |
| 📗 **拡張編** | 1565 語 | NEW HORIZON データベース（`new_horizon.xlsx`）から、マイナー過ぎる語を除いて収録 |

- 基本編と拡張編を**重複なし**で合算して、ちょうど **2000 語**になるよう調整しています。
- 拡張編は重要度（発信語・基本語・学年・初出語義など）の高い順に採用し、
  あふれた最もマイナーな語（ニッチな連語など）を除外して 2000 にそろえています。

## 品詞カテゴリ（細分化）

名詞／動詞／形容詞／副詞／**代名詞**／**疑問詞**／前置詞／接続詞／助動詞／冠詞／間投詞／短縮形／連語

- ご要望どおり **代名詞と疑問詞は分離**。疑問詞（what / who / where / when / why / how / which / whose / what time / how many …）は
  代名詞・副詞から取り出して独立カテゴリにしています。

## 主な機能

- **品詞ごとに挑戦** … カテゴリを選んで 1 セット 20 語ずつ出題。
- **クリアフラグ** … 「✓クリア！」を押すと記録され、進み具合（◯/◯語）が残ります。
- **🔀 ぜんぶシャッフル** … 全品詞からランダムに出題。
- **🔄 再挑戦** … クリア済みの語をたまに自動で再出題（頻度は設定で変更可）。間違えるとクリアが外れて学習に戻ります。専用の復習モードもあります。
- **設定** … 出題の向き（意味→英語／英語→意味）、出題順、再挑戦頻度、英語の読み上げ（音声）、クリア状況のリセット。

## 使い方

1. このリポジトリをダウンロード（または clone）。
2. `index.html` をブラウザで開く。
   - スマホの場合は、ファイルをスマホに送って開くか、下記の GitHub Pages を使うと便利です。
3. モード → 品詞を選んで学習スタート。

### GitHub Pages で公開する場合
リポジトリの **Settings → Pages → Branch** を `claude/modest-brown-5xx75y`（または `main`）の `/ (root)` にすると、
`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開できます。

## 開発者向け

### 単語データの再生成
元の Excel（`data/source/`）から `data/words.js` を作り直します。

```bash
pip install openpyxl
python3 tools/build_words.py     # = npm run build
```

`tools/build_words.py` 内の `TARGET`（既定 2000）や重要度スコアを変えれば、語数や採用基準を調整できます。

### 動作テスト
```bash
npm install        # jsdom（テスト用）
npm test           # DOM フローのスモークテスト
```

### ファイル構成
```
index.html            画面
css/style.css         スタイル
js/app.js             アプリ本体（出題・クリア管理・シャッフル・再挑戦）
data/words.js         単語データ（自動生成）
data/source/*.xlsx    元データ（先生の単語集 / NEW HORIZON）
tools/build_words.py  単語データ生成スクリプト
tools/smoke_test.js   スモークテスト（jsdom）
tools/shots.js        スクリーンショット撮影（puppeteer-core・任意）
```
