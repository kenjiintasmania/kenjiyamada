# 英語 模擬テストアプリ（正進社 過去問ベース）

正進社の過去問（中2／中3 第1回・第2回）と **出題傾向・問題数を完全一致** させた
模擬テストの Web アプリです。季節・数字・名前・場所などの **細部だけを入れかえ**、
文章の方向性（問われ方）はそのままにしています。

- **リスニングは「放送文を読んで・文字の選択肢から選ぶ」シミュレーション方式**
  （音声がなくても練習できるよう、放送原稿を文字で表示します）。
- 採点ボタンで自動採点。英作文・日本語記述は模範解答を見ながら自己採点。
- 単一ページ・JavaScript 内蔵・**オフライン可**。GitHub Pages 等にそのまま設置できます。

## 使い方

`index.html` をブラウザで開く（または GitHub Pages で公開）だけです。

1. **中2 模擬テスト** … 過去問「2年 Aコース」に対応（大問1〜7）。
   大問6は X/Y コースの選択問題です。
2. **中3 模擬テスト** … `第1回`／`第2回` を選択（大問1〜5）。
3. **答えの単語テスト** … 中2／中3第2回を解くのに必要な英単語集（既存アプリを収録）。

各テスト下部の「採点する」で採点、「もう一度」でリセット、「印刷」で紙に出せます。

## 学年と過去問の対応

| アプリ | 過去問 | 大問構成 | 配点 |
|---|---|---|---|
| 中2 模擬テスト | 2年 No.2 Aコース | 大問1〜7（リスニング／読解／対話／英作文／並べかえ X・Y／対話補充） | 100点 |
| 中3 第1回 模擬テスト | 3年 第1回 | 大問1〜5（リスニング 問題A〜D／チラシ読解／日記英作文／話し合い／意見文） | 100点 |
| 中3 第2回 模擬テスト | 3年 第2回 | 大問1〜5（リスニング 問題A〜D／メニュー読解／日記英作文／話し合い／原稿読解） | 100点 |

※ 過去問の表紙（「2年・No.2・Aコース」、注意書きの「大問6はX/Yコース選択」）が
X/Yコースを持つ大問1〜7の問題冊子と一致するため、それを **中2**、残り2回ぶんを
**中3 第1回・第2回** と判定しています。中3第2回のリスニングが「曜日」、単語集が
「四季・曜日・不規則動詞・動名詞・How系・文型・助動詞」と対応することも確認済みです。

## ファイル構成

```
index.html          トップ（中2 / 中3 / 単語集の入口）
chu3.html           中3の回（第1回 / 第2回）選択
exam.html           模擬テスト共通ランナー（?id=chu2 / chu3_1 / chu3_2）
assets/style.css    共通スタイル
assets/engine.js    描画＆自動採点エンジン
data/chu2.js        中2 模擬テスト データ
data/chu3_1.js      中3 第1回 模擬テスト データ
data/chu3_2.js      中3 第2回 模擬テスト データ
vocab_chu2.html     中2 答えの単語テスト（全36問）
vocab_chu3_2.html   中3 第2回 答えの単語テスト（全60問）
```

## 問題タイプ（engine.js）

`mcq`（記号選択・単一）／`mcqMulti`（複数選択）／`fill`（英単語・英文の空所補充）／
`bankpick`（共通選択肢から選ぶ）／`wordorder`（並べかえ）／`en`・`jp`（英作文・記述：模範解答で自己採点）。

問題内容を変えたいときは `data/*.js` を編集するだけで、表示・採点が自動で更新されます。
