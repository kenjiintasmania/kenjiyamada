# manuscript — 原稿の置き方と記法

## 置き方

- 1作品 = 1フォルダ。フォルダ名（スラッグ）は半角英数: `manuscript/<スラッグ>/genko.md`
- 例: `manuscript/sample/genko.md`

## 推奨記法（EPUB化スレが機械的に変換できる形）

```markdown
# 作品タイトル

## 1
パラグラフ本文…

- 選択肢の文 → 2
- 選択肢の文 → 3

## 2
選択肢のない強制ジャンプは文末に矢印だけ書く。 → 4

## 5
【バッドエンド】ジャンプのないパラグラフは必ずエンディングであることを本文に明記する。
```

- **パラグラフ**: `## 数字` で開始（番号は飛び飛びでもよいが、欠番参照は不可）
- **ジャンプ**: `→ 数字`（EPUB化の際、すべて内部リンクになる）
- **エンディング**: ジャンプのないパラグラフは【ハッピーエンド】【バッドエンド】等を明記

## 記法どおりでなくてもよい

手書きメモ・別形式の原稿でもそのまま置いてよい。EPUB化スレが解釈して変換し、
曖昧な箇所（欠番・ジャンプ先不明など）は勝手に直さず先生に確認する。
